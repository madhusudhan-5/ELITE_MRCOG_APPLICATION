# Elite MRCOG Admin Portal Guide

This guide provides step-by-step instructions for staff on how to manage course content and subscriptions.

## 1. Managing Course Modules
Learning Modules are the main containers for educational content.

1.  Navigate to **Manage Courses** in the sidebar.
2.  Select the **Part** (e.g., Part 3) from the tabs.
3.  Click **+ Create Module** once to add a new category.
4.  **Fields**:
    *   **Part**: Select the curriculum part.
    *   **Title**: The name of the module (e.g., "Communication Skills").
    *   **Thumbnail**: Upload a clear image representing the topic.
    *   **Short Description**: A brief summary shown to users on the dashboard.
5.  Click **Save Changes**.

## 2. Managing Reading Library & Stations
Once a module exists, you can add detailed reading articles and PDF stations.

### Adding a Reading Article
1.  Navigate to **Reading Library** or click **Manage Content** on a module card.
2.  Click **+ New Article**.
3.  **Fields**:
    *   **Module**: Select the parent module.
    *   **Title**: The name of the article.
    *   **Type**: Choose between "Course Material" or "Easy Read".
    *   **Short Description**: Summary shown on the card.
    *   **Overview Text**: Detailed syllabus/overview.
4.  Click **Save Article**.

### Adding Stations (PDFs)
Articles contain one or more "Stations". Each station typically holds a PDF file.
1.  In the **Reading Library** list, click on an **Article title** to expand its stations.
2.  Click **+ Add Station**.
3.  **Fields**:
    *   **Station Title**: Name of the station (e.g. "Mock Test 1").
    *   **Page Count**: Total pages in the PDF.
    *   **PDF File**: Upload the PDF document.
    *   **Free Perspective**: Check if this station should be available as a free preview.
4.  Click **Save Station**.

## 3. Managing Subscriptions & Bundles
Control user access and view financial transactions.

### Creating a Bundle
1.  Navigate to **Subscriptions** and select the **Bundles** tab.
2.  Click **+ New Bundle**.
3.  **Features**:
    *   Select which libraries are included (Reading, Video, Mock Exam).
    *   Set the **Price** and **Duration** (in days).
    *   Mark as **Featured** to highlight it as "Best Value".

### Viewing User Subscriptions
1.  In the **Subscriptions** page, use the **Subscriptions** tab.
2.  Search for users by email.
3.  You can manually **Activate** or **Cancel** a user's subscription from the actions column.

### Monitoring Payments
1.  Select the **Payments** tab to view all transaction logs.
2.  Filter by **Stripe** or **Pakistan** gateway to track successful or pending payments.

---
**Note**: All changes are reflected in real-time on the student dashboard. Ensure modules and articles are set to **Active** (Published) for them to be visible to students.
